    This project is now published in Nature Communications. You can find the code used to run the analyses and simulations, and the pre-registration as well as slides of conference talks and posters on our OSF page.

  • Genetics of non-cognitive skills

    This project is now published in Nature Genetics. Here you can find the FAQ, the tutorial to run your own GWAS-by-subtraction, the code used to run the analyses, and the summary statistics for the GWAS of non-cognitive and cognitive skills. You can also watch the 15-min talk I gave on this project at EARLI 2021.
